*******************************************
*****     Rules for the Gemz Game     *****
*******************************************

	1. There are 5 levels to the game. 
	2. The last level is impossible.
	3. Drag and release gems for matches.
	4. Gems can only be swapped in a 
	   north, east, west, and south manner.
	5. Game is won when you beat the last 
	   level.
